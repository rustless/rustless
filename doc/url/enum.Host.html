pub enum Host {
    Domain(String),
    Ipv6(Ipv6Address),
}

The host name of an URL.

Variants

Domain

A (DNS) domain name or an IPv4 address.

FIXME: IPv4 probably should be a separate variant. See https://www.w3.org/Bugs/Public/show_bug.cgi?id=26431

Ipv6

An IPv6 address, represented inside [...] square brackets so that : colon characters in the address are not ambiguous with the port number delimiter.
